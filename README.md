# TP ICT 301 - SOLID

NOM: DJIONZE NGOGANG

PRENOM: ANGE BLONDEL

MATRICULE: 25G2066


Les pricipes SOLID sont des règles de conception en programmation orientée objet qui permettent de produire des logiciels 
plus maintenables,flexibles et évolutifs.

Chaque principe vise à réduire le couplage,ameliorer la lisibilité du code et facilter les modiffication futures sans 
casser l'existants.
