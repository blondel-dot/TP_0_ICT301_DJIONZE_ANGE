# TP ICT 301 - SOLID

NOM: DJIONZE NGOGANG

PRENOM: ANGE BLONDEL

MATRICULE: 25G2066


Les principes SOLID sont des règles de conception en programmation orientée objet qui permettent de produire des logiciels 
plus maintenables,flexibles et évolutifs.

Chaque principe vise à réduire le couplage, améliorer la lisibilité du code et facilter les modifications futures sans 
casser l'existants.
Les 5 principes SOLID sont:
- SRP (Single Responsibility Principle);
- OCP ( Open Closed Principle);
- LSP ( Liskov Substitution Principle);
- ISP ( Interface Segregation Principle);
- DIP ( Dependency Inversion Principle)
